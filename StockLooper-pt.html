# Visão Geral

A simulação de Monte Carlo é um modelo de previsão que usa amostragem aleatória repetida para estimar a distribuição de probabilidade de diferentes resultados.

# Comece Agora

Teste os cenários padrão para entender como o sistema reage a diferentes condições.
* Ajuste os sliders com suas suposições de mercado; cada execução projetará o preço da ação para os próximos 5 anos.
* O painel "Variação da Ação" exibirá a oscilação total de cada execução.

# Funcionalidades Principais

* Identifique tendências de mercado: após várias execuções, uma "nuvem" de projeções se formará no gráfico, revelando uma tendência visual de possíveis faixas de preço, ajudando a definir o melhor momento de negociação.
* Visualize a volatilidade: uma ação estável apresentará uma linha densa, enquanto uma com beta elevado mostrará uma nuvem mais ampla de valores possíveis.
